Streamline Practice with Presets

TipsSettings

What Are Presets?

Manually configuring the same scale settings every time is tedious.

With presets, you can save your favorite settings and recall them with a single tap.

What Gets Saved in a Preset

Presets store all of these settings:

SettingExample
Scale TypeFive Tone, Rossini Scale, etc.
Starting PitchC4, G3, etc.
Tempo1.0 sec, 1.5 sec, etc.
Key Progression PatternAscending Then Descending, Ascending Only, etc.
Ascending Count3, 5, etc.
Descending Count0, 2, etc.
Ascending Step1 semitone, 2 semitones, etc.
Descending Step1 semitone, 2 semitones, etc.

How to Save a Preset

  1. Configure your desired settings on the recording screen
  2. Tap the “Save Preset” button in the settings panel
  3. Enter a name and save

Choose a descriptive name to find it easily later.

How to Load a Preset

  1. Tap “Presets” in the recording screen settings panel
  2. Select the preset you want to use
  3. Settings are applied automatically

Preset List Features

Sorting Options

Presets can be sorted in three ways:

SortDescription
Last UpdatedMost recently used at top (default)
NameAlphabetical order
PitchBy starting pitch (low to high)

Edit and Delete

You can edit or delete any preset from the preset list.

Recommended Preset Examples

For Warm-ups

  • Name: “Morning Warm-up”
  • Scale: Five Tone
  • Starting Pitch: A comfortable low note (e.g., C3)
  • Key Progression: Ascending Then Descending
  • Ascending Count: 3

For High Note Training

  • Name: “High Range Training”
  • Scale: Octave Repeat
  • Starting Pitch: Mid-range note (e.g., E3)
  • Key Progression: Ascending Only
  • Ascending Count: 5-6

For Low Range Development

  • Name: “Low Range Exercise”
  • Scale: Five Tone Down
  • Starting Pitch: Mid-range note (e.g., G3)
  • Key Progression: Descending Only
  • Descending Count: 4

For Advanced Practice

  • Name: “Full Range Workout”
  • Scale: Rossini Scale
  • Starting Pitch: Low note (e.g., C3)
  • Key Progression: Ascending Only
  • Ascending Count: 6-8

Tips for Using Presets

Create a Practice Routine

Use multiple presets in sequence to build a complete practice routine:

  1. “Morning Warm-up” to get your voice ready
  2. “High Range Training” to develop upper notes
  3. “Low Range Exercise” to maintain lower notes

Adjust Based on Daily Condition

If high notes feel difficult today, create a new preset with fewer ascending steps. Adjust presets as your voice changes and improves.

Use Descriptive Names

Instead of “C3 Start”, try “Morning Warm-up Light” - names that describe the purpose are more helpful.

Summary

  • Presets eliminate repetitive setup work
  • Scale type, pitch, key progression - everything is saved
  • Create purpose-specific presets to boost practice efficiency
  • Use clear, descriptive names for easy management
  • Combine multiple presets to build complete practice routines
