スケール音源の選び方

Tips設定

スケール音源とは

スケール練習中に再生されるガイド音の音色です。

好みや練習目的に合わせて、8種類の音源から選べます。

8種類の音源

ピアノ系

アコースティック・グランドピアノ（デフォルト）

クラシックなピアノの音色。

  • 特徴: クリアで聞き取りやすい
  • 向いている練習: オールラウンド
  • おすすめ度: ★★★★★

迷ったらこれを選びましょう。音程が明確で、どんな練習にも使えます。

エレクトリックピアノ

柔らかいエレピの音色。

  • 特徴: 温かみのある音
  • 向いている練習: ポップス、ジャズ
  • おすすめ度: ★★★★☆

リラックスした雰囲気で練習したい時に。

弦楽器系

アコースティックギター

ナイロン弦ギターの音色。

  • 特徴: 自然で優しい音
  • 向いている練習: 弾き語り練習
  • おすすめ度: ★★★☆☆

ギター伴奏で歌う曲の練習に最適です。

打楽器系

ビブラフォン

金属的で響きのある音色。

  • 特徴: 余韻が長い
  • 向いている練習: 音程感覚の訓練
  • おすすめ度: ★★★☆☆

サスティンが長いので、音程を合わせる練習に。

マリンバ

木の温かみのある音色。

  • 特徴: アタック明確、減衰早め
  • 向いている練習: リズム感を重視した練習
  • おすすめ度: ★★★☆☆

テンポ感をつかみたい時に。

管楽器系

フルート

息の流れを感じる音色。

  • 特徴: 繊細で透明感のある音
  • 向いている練習: ブレスコントロール
  • おすすめ度: ★★★★☆

管楽器のようなフレージングを意識した練習に。

クラリネット

暖かく丸みのある音色。

  • 特徴: 中音域が豊か
  • 向いている練習: 中音域の安定
  • おすすめ度: ★★★☆☆

中音域を中心に練習する時に。

その他

サイン波

純粋な正弦波の音。

  • 特徴: 倍音がなく、基音のみ
  • 向いている練習: 純粋な音程訓練
  • おすすめ度: ★★★☆☆

音程を純粋に確認したい時に。音楽的な響きはありませんが、ピッチの確認には最適です。

目的別おすすめ音源

練習目的おすすめ音源
基本練習・迷ったらグランドピアノ
リラックスしたいエレクトリックピアノ
弾き語り練習アコースティックギター
音程感覚の訓練ビブラフォン、サイン波
ブレス練習フルート
テンポ感をつかむマリンバ

設定方法

  1. ホーム画面から「設定」をタップ
  2. 「オーディオ」セクションの「出力設定」を開く
  3. 「スケール音源」から好みの音色を選択

音量調整

スケール音の音量は、「スケール再生音量」スライダーで調整できます。

  • 大きめ: ガイド音をしっかり聞きたい時
  • 小さめ: 自分の声を中心に聞きたい時

録音再生音量とは別に設定できるので、練習時と確認時で使い分けられます。

まとめ

  • 迷ったらグランドピアノが最も万能
  • 練習目的に合わせて音源を変えると効果的
  • フルートはブレス練習に、ビブラフォンは音程練習に
  • 音量も合わせて調整して、最適な練習環境を作りましょう
