# Vocalis Studio ブランドボイスガイドライン

## キャラクター設定

### 基本トーン
- 丁寧で親切、しかし過度にフレンドリーにならない
- 事務的だが冷たくない「信頼できる専門家」の印象
- 高精度・高品質なプロダクトイメージに合致

### ペルソナイメージ
- 声楽や発声に詳しい専門家
- 質問には的確に答える
- 押し付けがましくない

---

## 言葉遣いの指針

### OK / NG 対照表

| OK | NG |
|----|----|
| 〜できます | 〜しちゃおう！ |
| 〜をお試しください | 〜してみて！ |
| ご確認ください | チェックしてね |
| 効果的です | めっちゃ良い |
| ぜひご活用ください | 使ってみて！ |
| ご設定ください | 設定してね |
| 〜が可能です | 〜できちゃう |

### 文体ルール
- 「です・ます」調で統一
- 専門用語は使用可（ピッチ、スケール、BPMなど）
- 簡潔に要点を伝える（箇条書き活用）
- 1文は短めに、読みやすさを重視

---

## 絵文字の使用ガイド

### 推奨される使い方
- 機能説明のアイコンとして使用
- 1投稿あたり5〜8個程度を目安に

### 機能系アイコン（積極使用）
- 🎤 録音・マイク関連
- 🎵 音楽・スケール関連
- 🎹 ピッチ・キー関連
- ⚙️ 設定関連
- 📊 分析・グラフ関連
- 🎙️ 入力設定
- 🔊 出力設定
- 🌐 言語設定
- ⏱️ テンポ関連
- 📈 進行・上昇関連

### 控えめに使用
- ✨ 強調したい場合のみ

### 避けるべき絵文字
- 😆 😊 🎉 など感情表現が強いもの
- 🔥 💪 など煽り系
- 顔文字全般

## 締めの一文 例

- ご自身の声域や練習目的に合わせてご設定ください。
- 録音前にご確認いただくと、より正確な分析結果が得られます。
- ぜひご活用ください。
- 詳細は設定画面からご確認いただけます。
- お試しいただければ幸いです。

※「ぜひお試しください！」のように「！」は控えめに使用
