# リリース投稿テンプレート

## 基本情報
- **カテゴリ**: リリース報告
- **頻度**: 随時（リリース時）

---

## 文字数制限

| プラットフォーム | 制限 | 備考 |
|------------------|------|------|
| X（Twitter） | 280文字 | 日本語は140文字相当、URLは23文字としてカウント |
| Instagram | 2,200文字 | 改行で読みやすく |

---

## X（Twitter）

**文字数目安**: 本文100〜120文字 + URL(23文字) + ハッシュタグ(約40文字) = 約180文字以内

```
🎵 Vocalis Studio v[VERSION]をリリース

・[変更点1]
・[変更点2]
・[変更点3]

App Store: [URL]

#VocalisStudio #ボイトレ #音声分析
```

## Instagram

**キャプション**:
```
Vocalis Studio version [VERSION] を公開しました。

[主な変更点の説明]

詳しくはApp Storeをご覧ください。

#VocalisStudio #VoiceTrainingApp #ボイストレーニング #AppUpdate
```

**画像案**:
- アプリアイコン + バージョン番号
- 新機能のスクリーンショット
- Before/After比較（UI改善時）
