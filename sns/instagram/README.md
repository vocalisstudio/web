# Instagram運用

## 画像仕様

| 種類 | サイズ | アスペクト比 |
|------|--------|--------------|
| フィード（正方形） | 1080x1080px | 1:1 |
| フィード（縦長） | 1080x1350px | 4:5（推奨） |
| ストーリー/リール | 1080x1920px | 9:16 |
| カルーセル | 1080x1080px | 1:1 |

## 投稿ガイドライン

- キャプション: 最大2,200文字
- ハッシュタグ: 最大30個（5〜10個推奨）
- 改行で読みやすく整形

## 固定ハッシュタグ

### 日本語
```
#VocalisStudio #VoiceTrainingApp #ボイストレーニング
```

### 追加候補（日本語）
```
#音声分析 #ボイトレアプリ #AppUpdate #iOSアプリ
#歌練習 #声楽 #ナレーション #ボーカルトレーニング
```

### English
```

```

### Additional (English)


## ディレクトリ構成

```
instagram/
├── posts/
│   └── YYYY/MM/
│       ├── ja/          # 日本語投稿
│       └── en/          # 英語投稿
└── assets/
    └── images/
        ├── ja/          # 日本語用画像
        └── en/          # 英語用画像
```

## デザイン方針

- 背景: 白または淡いグレー基調
- スクリーンショット中心
- 余白を十分に取る
- 統一感のあるテンプレート使用
