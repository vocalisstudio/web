# X（Twitter）運用

## 画像仕様

| 種類 | サイズ | アスペクト比 |
|------|--------|--------------|
| 単一画像 | 1200x675px | 16:9（推奨） |
| 複数画像 | 1200x675px | 16:9 |
| カード画像 | 800x418px | 1.91:1 |

## 投稿ガイドライン

- 文字数: 最大280文字（日本語は140文字相当）
- ハッシュタグ: 2〜3個推奨
- リンク: 23文字としてカウント

## 固定ハッシュタグ

```
#VocalisStudio #ボイトレ #音声分析
```

## ディレクトリ構成

```
x/
├── posts/           # 投稿アーカイブ
│   └── YYYY/MM/
└── assets/
    └── images/      # X用画像素材
```
