## キー進行とは

スケールを1回演奏した後、自動的にキー（調）を変えて次のスケールを演奏する機能です。

たとえば、C4から始めて半音ずつ上げていくと、C4 → C#4 → D4 → D#4... と続きます。

これにより、一度の録音で複数のキーを効率的に練習できます。

## 4つの進行パターン

### 上昇のみ（Ascending Only）

キーが上がっていくだけのパターンです。

**例**: C4 → C#4 → D4 → D#4（3回上昇の場合）

高音域を少しずつ広げていく練習に最適です。

### 下降のみ（Descending Only）

キーが下がっていくだけのパターンです。

**例**: C4 → B3 → Bb3 → A3（3回下降の場合）

低音域の開発や、リラックスした発声練習に向いています。

### 上昇→下降（Ascending Then Descending）

まず上昇し、最高点から下降するパターンです。**デフォルト設定**です。

**例**: C4 → C#4 → D4 → D#4 → D4 → C#4 → C4

ウォームアップに理想的。徐々に高音域に挑戦し、クールダウンしながら戻ってきます。

### 下降→上昇（Descending Then Ascending）

まず下降し、最低点から上昇するパターンです。

**例**: C4 → B3 → Bb3 → A3 → Bb3 → B3 → C4

低音域から始めたい場合や、変化をつけた練習に。

## キー変更の設定項目

### 上昇回数・下降回数

それぞれ0〜24回まで設定できます。

| 設定 | デフォルト | 説明 |
|------|-----------|------|
| 上昇回数 | 3回 | キーを上げる回数 |
| 下降回数 | 0回 | キーを下げる回数 |

デフォルトでは「上昇→下降」パターンで上昇3回なので、計7回のスケール演奏になります（C→C#→D→D#→D→C#→C）。

### ステップ間隔

キーをいくつずつ変えるかを設定します。

| 間隔 | 説明 | 例（C4から上昇） |
|------|------|------------------|
| 1（半音） | デフォルト | C4 → C#4 → D4 |
| 2（全音） | 2半音ずつ | C4 → D4 → E4 |
| 3（短3度） | 3半音ずつ | C4 → D#4 → F#4 |
| 12（オクターブ） | 1オクターブずつ | C4 → C5 → C6 |

## 実践的な設定例

### 初心者向け：ゆるやかな上昇

- **パターン**: 上昇のみ
- **上昇回数**: 2〜3回
- **ステップ**: 1（半音）

少しずつ高音に慣れていくのに最適。無理のない範囲で練習できます。

### ウォームアップ：往復練習

- **パターン**: 上昇→下降
- **上昇回数**: 3回
- **ステップ**: 1（半音）

デフォルト設定そのままでOK。毎日の練習の始まりに。

### 音域拡張：広いレンジ

- **パターン**: 上昇のみ
- **上昇回数**: 6〜8回
- **ステップ**: 1（半音）

高音域の限界に挑戦。上昇回数を増やして、どこまで出せるか確認します。

### 効率練習：大きなステップ

- **パターン**: 上昇→下降
- **上昇回数**: 4回
- **ステップ**: 2（全音）

全音ずつ上げることで、時間を短縮しながら広い音域をカバーします。

### 低音開発：下降フォーカス

- **パターン**: 下降のみ
- **下降回数**: 4回
- **ステップ**: 1（半音）

低音域を広げたい方に。Five Tone Downスケールとの組み合わせがおすすめ。

## 開始ピッチの選び方

キー進行と合わせて、開始ピッチも重要です。

| 声種（目安） | 開始ピッチ例 | 上昇回数の目安 |
|-------------|------------|---------------|
| 低め（バス〜バリトン） | A2〜C3 | 4〜6回 |
| 中程度（テノール〜アルト） | C3〜E3 | 3〜5回 |
| 高め（メゾ〜ソプラノ） | E3〜G3 | 3〜4回 |

自分の楽に出せる低めの音から始めて、上昇で高音域に挑戦するのが基本です。

## キー進行使用時のヒント

### 無理をしない

高音で声が出なくなったら、次回は上昇回数を減らすか、開始ピッチを下げましょう。

### 最高音を記録する

「今日はD5まで出た」など、到達した最高音をメモしておくと、成長が実感できます。

### スケールタイプとの組み合わせ

- **Five Tone + 上昇→下降**: 基本のウォームアップ
- **Octave Long Tone + 上昇のみ**: 高音の安定練習
- **Rossini Scale + 上昇のみ**: 上級者の音域拡張
