## ビブラートとは

ビブラートは、ピッチを周期的に揺らす歌唱技法です。

適切なビブラートは歌声に豊かさと表現力を加えます。Vocalis Studioでは、ビブラートを3つの指標で分析します。

## 3つの分析指標

### レート（Rate）

ビブラートの**速さ**を表します。単位はHz（1秒あたりの揺れの回数）。

| 値 | 評価 |
|----|------|
| 4〜5 Hz | ゆったりとした印象 |
| 5〜6 Hz | 一般的な範囲 |
| 6〜7 Hz | やや速め |
| 7 Hz以上 | 速い（緊張気味の可能性）|

**目安**: 5〜6 Hzが多くのジャンルで心地よいとされています。

### 振幅（Extent）

ビブラートの**幅**を表します。単位はセント（cents）。

| 値 | 評価 |
|----|------|
| 15〜30 cents | 控えめ |
| 30〜50 cents | 標準的 |
| 50〜80 cents | 広め |
| 80 cents以上 | 非常に広い |

**目安**: 30〜50 centsが自然に聞こえる範囲です。

### 規則性（Regularity）

ビブラートの**安定性**を表します。パーセンテージで表示。

| 値 | 評価 |
|----|------|
| 80%以上 | 非常に安定 |
| 60〜80% | 良好 |
| 40〜60% | 改善の余地あり |
| 40%未満 | 不安定 |

**目安**: 60%以上を目指しましょう。

## 統計画面での確認

録音の統計情報で、ビブラート分析の結果を確認できます。

![統計情報画面](/images/ja/05_statistics_pitch.png)

- **平均レート**: ビブラート全体の平均速度
- **平均振幅**: ビブラート全体の平均幅
- **平均規則性**: ビブラート全体の安定度

## よくあるパターンと改善方法

### レートが速すぎる（7Hz以上）

**原因**: 喉に力が入っている、緊張している

**改善方法**:
- リラックスした状態で歌う
- ゆっくりとした曲で練習
- 腹式呼吸を意識する

### レートが遅すぎる（4Hz未満）

**原因**: 息の支えが弱い、意識的にかけすぎている

**改善方法**:
- 自然に声を伸ばす練習
- 意識的なビブラートではなく、自然に出るのを待つ

### 振幅が狭すぎる

**原因**: 緊張、喉の固さ

**改善方法**:
- Octave Long Toneスケールで長く伸ばす練習
- リラックスして息を流す

### 振幅が広すぎる

**原因**: コントロール不足、意識的にかけすぎ

**改善方法**:
- ストレートトーン（ビブラートなし）の練習
- 振幅を意識的にコントロールする練習

### 規則性が低い

**原因**: ビブラートがまだ安定していない

**改善方法**:
- 一定のテンポで揺らす練習
- メトロノームに合わせたビブラート練習

## ビブラート練習のコツ

### Octave Long Toneを活用

4拍間トップノートをホールドするスケールです。長く伸ばすことで、自然なビブラートが出やすくなります。

### 最初はストレートで

ビブラートをかけようとせず、まっすぐ伸ばす練習から始めましょう。安定した発声ができてから、自然にビブラートが加わります。

### 録音して確認

自分のビブラートを客観的に確認することが大切です。統計情報を見て、改善点を把握しましょう。

## ジャンル別の傾向

| ジャンル | レート | 振幅 |
|---------|--------|------|
| クラシック | 5〜6 Hz | 広め（50〜80 cents） |
| ポップス | 5〜7 Hz | 標準的（30〜50 cents） |
| ジャズ | 様々 | 表現に応じて変化 |
| ロック | 速め | 控えめ〜標準 |

ジャンルによって「良いビブラート」は異なります。自分が目指すスタイルに合わせて練習しましょう。

## まとめ

- **レート**: 5〜6 Hzが一般的な目安
- **振幅**: 30〜50 centsが自然な範囲
- **規則性**: 60%以上を目標に
- 無理にかけようとせず、リラックスが基本
- Octave Long Toneスケールが練習に効果的
