## 数値で見る歌唱

ピッチグラフやスペクトログラムは視覚的な確認に向いていますが、「どのくらい正確か」を客観的に把握するには数値が役立ちます。

統計情報シートでは、録音を分析した結果を数値で確認できます。

![統計情報（ピッチ分析）](/web/blog/images/ja/05_statistics_pitch.png)

## 統計情報でわかること

- 音程の正確さ（全体・各音）
- シャープ/フラットの傾向
- ピッチの安定度
- ビブラートの特徴
- 声の高周波成分（Premium）

## ピッチ分析セクション

### Overall（全体）

| 項目 | 意味 |
|------|------|
| accuracy | 全体の正確度（%）。目標音程にどれだけ近かったか |
| medianCents | 中央セント偏差。全体的にシャープ寄りかフラット寄りか |
| range | ピッチ範囲。録音中に出した音域の幅 |

accuracyが高いほど、目標音程に正確に歌えていたことを示します。

### Position（ポジション）

| 項目 | 意味 |
|------|------|
| inTuneRate | 合っている率（%）。目標音程に収まっていた時間の割合 |
| sharpRate | シャープ率（%）。高めにズレていた時間の割合 |
| flatRate | フラット率（%）。低めにズレていた時間の割合 |
| averageDeviation | 平均偏差（cents）。平均的なズレ幅 |

自分がシャープしやすいかフラットしやすいかを把握するのに役立ちます。

### Pitch（ピッチ詳細）

| 項目 | 意味 |
|------|------|
| noteOnsetAccuracyScore | 音の立ち上がり正確度。各音の出だしの正確さ |
| pitchStabilityScore | ピッチ安定度。音を伸ばしている間の安定性 |
| intervalAccuracyScore | 音程正確度。音から音への移動の正確さ |

これらのスコアから、どの要素を改善すべきかがわかります。

## ビブラート分析

ビブラートがかかっている部分を検出し、その特徴を数値化しています。

| 項目 | 意味 |
|------|------|
| averageRate | 平均レート（Hz）。ビブラートの速さ |
| averageExtent | 平均振幅（cents）。ビブラートの揺れ幅 |
| averageRegularity | 平均規則性（%）。ビブラートの均一さ |

一般的に、自然なビブラートは5〜7Hz程度のレートで、規則的なものが好まれます。

## スペクトル分析（Premium）

高周波成分の特徴を分析します。この機能はPremium会員向けです。

![統計情報（スペクトル分析）](/web/blog/images/ja/05_statistics_spectrum.png)

| 項目 | 周波数範囲 | 意味 |
|------|----------|------|
| Singer's Formant (SF) | 2.5-3.5 kHz | シンガーズフォルマント。声の通りやすさに関係 |
| Brightness | 4-6 kHz | 明るさ。声の明瞭さに関係 |
| Airiness | 6-9 kHz | 空気感。息の混じり具合に関係 |

これらは声質の特徴を客観的に把握するための参考指標です。

## 活用のヒント

### 傾向を把握する

一回の録音だけでなく、複数の録音を比較することで自分の傾向が見えてきます。「いつもフラット気味」「高音で不安定になりやすい」といった発見につながります。

### 改善点を絞る

すべての指標を同時に改善しようとせず、まずは一つに絞るのが効果的です。たとえば「音の立ち上がり」に課題があるなら、そこに集中して練習してみてください。

### グラフと併用する

統計情報で傾向を把握し、ピッチグラフで具体的な箇所を確認する。この組み合わせが効率的な振り返りにつながります。

## まとめ

- 統計情報は歌唱を数値で客観的に評価できる
- ピッチ分析で正確度・安定度・傾向がわかる
- ビブラート分析でビブラートの特徴を把握できる
- スペクトル分析（Premium）で声質の特徴がわかる
- 複数の録音を比較して傾向を把握するのがおすすめ
