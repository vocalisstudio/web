# プロモーション画像4: デザイン指示書

## 1. 基本情報
- **テーマ:** プロレベルの分析機能
- **キャッチコピー:** プロが聴き分ける「響き」まで。
- **Catchphrase (EN):** Analyze What Pros Hear.
- **使用スクリーンショット:** `web/blog/public/images/ja/05_statistics_spectrum.png`

## 2. デザイン要件
- **画像サイズ:** 1242 x 2688 px
- **全体的な雰囲気:** 専門的、高品質、ディープ

## 3. レイアウトと構図
- **背景:**
    - これまでとは少し変化をつけ、**深い紺色（`#1E3A5F` テキストカラー）**からプライマリカラー（`#4A9FD8`）へのグラデーションを使用し、専門性を表現する。
    - 薄い幾何学模様は同様に配置する。
- **デバイス（iPhoneモックアップ）:**
    - **画面の左側**に配置する。
    - スクリーンショット内の「**Spectrum Analysis**」セクション、特に「**SF (Singer's Formant)**」の項目がはっきりと見えるようにする。
- **テキスト（キャッチコピー）:**
    - **「プロが聴き分ける「響き」まで。」**を、画面右側の空いたスペースに配置する。
- **テキスト（補足）:**
    - **「声の明るさや力強さに関わる「シンガーズフォルマント」を分析。より豊かな声を目指すあなたへ。」**をキャッチコピーの下に配置する。
    - **(EN):** "Measure Singer's Formant—the resonance behind vocal projection."
- **装飾:**
    - 「**シンガーズフォルマント**」という単語の近くに、周波数スペクトルを模した抽象的なラインアートを薄く配置し、専門的な雰囲気を補強する。

## 4. タイポグラフィ
- **フォントファミリー:** ヒラギノ角ゴ ProN、または同等のサンセリフ体。
- **キャッチコピー:**
    - フォントウェイト: **Bold (太字)**
    - フォントサイズ: 100pt （少し長めの文言に合わせて調整）
    - 色: `#FFFFFF` (白)
- **補足テキスト:**
    - フォントウェイト: **Regular (標準)**
    - フォントサイズ: 48pt
    - 色: `#FFFFFF` (白)
    - 行間: 1.5

## 5. 色彩設計
- **基本:** アプリのテーマカラーを基調としつつ、より深みのある配色を試みる。
- **背景:** 深い紺色 (`#1E3A5F`) からプライマリカラー (`#4A9FD8`)へのグラデーション。
- **テキスト:** `#FFFFFF` (白)。
- **装飾:** ラインアートには、白またはアクセントカラー (`#6DB8E8`) を使用する。
