# プロモーション画像2: デザイン指示書

## 1. 基本情報
- **テーマ:** スペクトログラム分析
- **キャッチコピー:** 倍音構造を、可視化する。
- **Catchphrase (EN):** See the Overtones.
- **使用スクリーンショット:** スペクトログラム表示画面（AnalysisViewのspectrogramタブ）

## 2. デザイン要件
- **画像サイズ:** 1242 x 2688 px
- **全体的な雰囲気:** 技術的、信頼感、精確

## 3. レイアウトと構図
- **背景:**
    - 1枚目と同様のグラデーション背景と、薄い幾何学模様を配置する。
- **デバイス（iPhoneモックアップ）:**
    - **画面の右側**に、やや斜めに（左に少し傾けて）配置し、動きを出す。
    - スクリーンショット内のスペクトログラム（時間×周波数の2Dカラーマップ）がはっきりと見えるようにする。
- **テキスト（キャッチコピー）:**
    - **「倍音構造を、可視化する。」**を、画面左側の空いたスペースに、デバイスと少し重なるようにして配置する。
- **テキスト（補足）:**
    - **「倍音の変化を時系列で確認。」**をキャッチコピーの下に配置する。
    - **(EN):** "Track harmonic changes over time."
- **装飾:**
    - キャッチコピーの背景に、スクリーンショットからスポイトした**アクセントカラー（`#3A8EC5`など）**を使い、円や四角などのシンプルな図形を透明度50%で薄く配置し、視線を誘導する。

## 4. タイポグラフィ
- **フォントファミリー:** ヒラギノ角ゴ ProN、または同等のサンセリフ体。
- **キャッチコピー:**
    - フォントウェイト: **Bold (太字)**
    - フォントサイズ: 120pt
    - 色: `#FFFFFF` (白)
- **補足テキスト:**
    - フォントウェイト: **Regular (標準)**
    - フォントサイズ: 48pt
    - 色: `#FFFFFF` (白)
    - 行間: 1.5

## 5. 色彩設計
- **基本:** アプリのテーマカラーを基調とする。
- **背景:** `#4A9FD8` グラデーション。
- **テキスト:** `#FFFFFF` (白)。
- **装飾:** スクリーンショット内のグラフ等で使われているアクセントカラー (`#3A8EC5`) を抽出し、背景装飾に使用して統一感を出す。
