# プロモーション画像5: デザイン指示書

## 1. 基本情報
- **テーマ:** 柔軟なカスタマイズ性
- **キャッチコピー:** あなたのレベルで、着実に上達。
- **使用スクリーンショット:** `web/blog/public/images/ja/02_recording_idle.png`

## 2. デザイン要件
- **画像サイズ:** 1242 x 2688 px
- **全体的な雰囲気:** 使いやすい、親しみやすい、体系的

## 3. レイアウトと構図
- **背景:**
    - 1枚目と同様の、プライマリカラー (`#4A9FD8`) を基調としたグラデーション背景に戻し、シリーズとしての統一感を保つ。
- **デバイス（iPhoneモックアップ）:**
    - 画面中央にまっすぐに配置する。
    - スクリーンショットは、「スケール設定」パネルが開いており、「**Scale Type**」「**Start Pitch**」「**Tempo**」などのカスタマイズ項目が見える状態にする。
- **テキスト（キャッチコピー）:**
    - **「あなたのレベルで、着実に上達。」**を、画面上部に配置する。
- **テキスト（補足）:**
    - **「豊富な練習メニューを、キーもテンポも自由自在。自分だけの課題で、無理なく続けられる。」**を、デバイスの下のスペースに配置する。
- **装飾:**
    - 左右の空いたスペースに、「9種類のスケール」「キー・テンポ自由設定」「プリセット保存」といった**アイコンと短いテキスト**を組み合わせたフィーチャーリストを3つほど配置し、カスタマイズ性の高さを具体的に示す。

## 4. タイポグラフィ
- **フォントファミリー:** ヒラギノ角ゴ ProN、または同等のサンセリフ体。
- **キャッチコピー:**
    - フォントウェイト: **Bold (太字)**
    - フォントサイズ: 110pt
    - 色: `#FFFFFF` (白)
- **補足テキスト:**
    - フォントウェイト: **Regular (標準)**
    - フォントサイズ: 48pt
    - 色: `#FFFFFF` (白)
    - 行間: 1.5
- **フィーチャーリスト:**
    - フォントウェイト: **Medium**
    - フォントサイズ: 36pt
    - 色: `#FFFFFF` (白)

## 5. 色彩設計
- **基本:** アプリのテーマカラーを基調とする。
- **背景:** `#4A9FD8` グラデーション。
- **テキスト:** `#FFFFFF` (白)。
- **装飾:** フィーチャーリストのアイコンは、白（`#FFFFFF`）でシンプルに作成する。
