# プロモーション画像5: デザイン指示書

## 1. 基本情報
- **テーマ:** 統計・詳細分析
- **キャッチコピー:** 技術課題を、数値で特定。
- **Catchphrase (EN):** Diagnose with Data.
- **使用スクリーンショット:** `web/blog/public/images/ja/05_statistics_pitch.png`（StatisticsSheetViewのピッチ分析セクション）

## 2. デザイン要件
- **画像サイズ:** 1242 x 2688 px
- **全体的な雰囲気:** 技術的、精確、専門的

## 3. レイアウトと構図
- **背景:**
    - 1枚目と同様の、プライマリカラー (`#4A9FD8`) を基調としたグラデーション背景に戻し、シリーズとしての統一感を保つ。
- **デバイス（iPhoneモックアップ）:**
    - 画面中央にまっすぐに配置する。
    - スクリーンショットは、統計情報シート（StatisticsSheetView）が表示された状態で、「**Overall Accuracy**」「**Pitch Stability**」「**Vibrato**」などの分析項目が見える状態にする。
- **テキスト（キャッチコピー）:**
    - **「技術課題を、数値で特定。」**を、画面上部に配置する。
- **テキスト（補足）:**
    - **「正確度・安定性・ビブラート。数値で診断。」**を、デバイスの下のスペースに配置する。
    - **(EN):** "Accuracy. Stability. Vibrato. Measured."
- **装飾:**
    - 左右の空いたスペースに、「音程正確度」「ピッチ安定性」「ビブラート分析」といった**アイコンと短いテキスト**を組み合わせたフィーチャーリストを3つほど配置し、分析項目の具体性を示す。

## 4. タイポグラフィ
- **フォントファミリー:** ヒラギノ角ゴ ProN、または同等のサンセリフ体。
- **キャッチコピー:**
    - フォントウェイト: **Bold (太字)**
    - フォントサイズ: 110pt
    - 色: `#FFFFFF` (白)
- **補足テキスト:**
    - フォントウェイト: **Regular (標準)**
    - フォントサイズ: 48pt
    - 色: `#FFFFFF` (白)
    - 行間: 1.5
- **フィーチャーリスト:**
    - フォントウェイト: **Medium**
    - フォントサイズ: 36pt
    - 色: `#FFFFFF` (白)

## 5. 色彩設計
- **基本:** アプリのテーマカラーを基調とする。
- **背景:** `#4A9FD8` グラデーション。
- **テキスト:** `#FFFFFF` (白)。
- **装飾:** フィーチャーリストのアイコンは、白（`#FFFFFF`）でシンプルに作成する。
