# プロモーション画像1: デザイン指示書

## 1. 基本情報
- **テーマ:** アプリの核心機能
- **キャッチコピー:** 歌声が、見える。
- **使用スクリーンショット:** `web/blog/public/images/ja/02_recording_in_progress.png`

## 2. デザイン要件
- **画像サイズ:** 1242 x 2688 px (6.5インチiPhone向け)
- **全体的な雰囲気:** クリーン、モダン、プロフェッショナル、音楽的

## 3. レイアウトと構図
- **背景:**
    - アプリのプライマリカラー (`#4A9FD8`) から、少し明るい青への、**左上から右下へ抜ける**緩やかなグラデーションを配置する。
    - 背景に、音の波形や五線譜をモチーフにした**薄いグレー（`#EAEAEA`）の幾何学模様**を、透明度20%でうっすらと重ね、単調さをなくす。
- **デバイス（iPhoneモックアップ）:**
    - 画面中央より少し下寄りに配置する。
    - デバイスの下には、**柔らかく広がるドロップシャドウ**（色: `#000000`, 透明度: 15%, ぼかし: 30px）を追加し、立体感を出す。
- **テキスト（キャッチコピー）:**
    - **「歌声が、見える。」**というテキストを、画面上部の最も目立つ位置に配置する。
- **テキスト（補足）:**
    - **「リアルタイムに音程を可視化。正しい音を、目で見て確認しながら練習できる。」**を、キャッチコピーの下に、少し小さめのサイズで配置する。

## 4. タイポグラフィ（フォント）
- **フォントファミリー:** ヒラギノ角ゴ ProN、またはそれに準ずるクリーンなサンセリフ体（例: Source Han Sans JP）。
- **キャッチコピー:**
    - フォントウェイト: **Bold (太字)**
    - フォントサイズ: 120pt
    - 色: `#FFFFFF` (白)
- **補足テキスト:**
    - フォントウェイト: **Regular (標準)**
    - フォントサイズ: 48pt
    - 色: `#FFFFFF` (白)
    - 行間: 1.5

## 5. 色彩設計
- **基本:** アプリのデザインシステムカラーを遵守する。
- **背景:** `#4A9FD8` を基調としたグラデーション。
- **テキスト:** `#FFFFFF` (白)。
- **装飾:** 背景の幾何学模様は、主張しすぎないよう薄いグレー（`#EAEAEA`など）を使用する。
